# Contributing to Consciousness Proxy

Thank you for your interest in contributing!

## How to Contribute

- Fork the repository and create your branch from `main`.
- If you've fixed a bug or added a feature, add tests if possible.
- Ensure code is clean, modular, and documented.
- Open a pull request with a clear description of your changes.

## Code Style
- Use Python 3.9+ typing and docstrings.
- Keep API endpoints RESTful and well-documented.
- Prefer small, focused commits.

## Issues
- Use GitHub Issues for bugs, feature requests, and questions.

## Community
- Be respectful and constructive.
- All contributions are welcome!
